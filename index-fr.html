Qu'est-ce que XVB ?

XVB (Extended Video Broadcasting) est un cœur de lecture web qui s'exécute entièrement côté client. Il est conçu pour gérer l'IPTV et les sources de type diffusion directement dans le navigateur, avec un accent sur la fiabilité, la compatibilité et la performance.

Ce projet n'est pas une "plateforme backend IPTV" — c'est un moteur de lecture ciblé. Vous apportez vos sources (playlists, flux ou fichiers) ; XVB gère la lecture.

Pourquoi c'est important.

  • Lecture stable de longue durée (direct ou VOD)
  • Initialisation et changement de flux rapides
  • Compatible avec les formats de playlist IPTV courants
  • Séparation claire entre le moteur de lecture et l'interface

Support du streaming.

Conçu pour les environnements IPTV et une lecture en direct résiliente sur réseaux instables.

Protocoles et playlists

XVB supporte plusieurs technologies de streaming couramment utilisées :

  • HLS (HTTP Live Streaming, .m3u8)
  • MPEG-DASH (.mpd)
  • MPEG-TS flux en direct (.ts)
  • M3U / M3U8 playlists
  • Flux standard MP4

Formats de médias jouables

Selon le support du navigateur, XVB peut lire les formats suivants :

  • Vidéo : m3u8, mp4, webm, ogv, ts
  • Audio : mp3, aac, m4a, ogg, opus, wav, flac

La disponibilité des codecs peut varier selon les navigateurs et les appareils.

Manuel d'Utilisation

Comprendre l'Interface

XVB dispose d'une barre d'outils minimaliste mais puissante, conçue pour la vitesse et la surveillance en temps réel. Voici le détail de chaque élément de votre tableau de bord.

Metriques Live

  • Compteur de Chaînes: Affiche le nombre total de flux lisibles actuellement chargés à partir de vos listes de lecture actives.
  • État de l'EPG: Horodatage en temps réel de la dernière mise à jour du Guide Électronique des Programmes (sources Italie et UK).
  • Utilisateurs en Ligne: Indique combien d'utilisateurs sont actuellement connectés à l'écosystème XVB dans le monde.

Navigation

  • Recherche Intelligente: Filtrage instantané. Tapez le nom d'une chaîne pour la trouver immédiatement sans défiler.
  • Cloud d'Échantillons: Si aucune liste n'est présente, cliquez ici pour charger une liste de test technique (MP4, AAC, MP3, OGG, MPD, TS et HLS).

Gestion des Listes

  • Ajouter URL Distante: Importez des liens M3U directs. Note: cette méthode cible les liens statiques directs pour une performance maximale.
  • Import Local: Parcourez votre appareil pour télécharger des fichiers .m3u ou .m3u8 stockés localement.
  • Vider le Cache: Efface instantanément toutes les listes mémorisées et les données du cache local de votre navigateur.

Astuce: Utilisez le bouton Vider le Cache si vous rencontrez des problèmes après la mise à jour d'un fichier local. XVB stocke les données côté client pour un démarrage ultra-rapide.

Playlists

Créez votre propre playlist M3U.

XVB charge des playlists depuis une URL ou un fichier local. Le format standard est M3U / M3U8, avec métadonnées (nom, logo, groupe) suivies de l'URL du flux.

Conseil : utilisez group-title pour les catégories et tvg-logo pour une interface plus visuelle.

Exemple M3U

Enregistrez sous ma-playlist.m3u (UTF-8). 

#EXTM3U

#EXTINF:-1 tvg-name="Chaîne Démo 1" tvg-logo="https://example.com/logo1.png" tvg-id="demo.channel.1" group-title="News",Chaîne Démo 1
https://example.com/stream/channel1.m3u8

#EXTINF:-1 tvg-name="Radio Démo" tvg-logo="https://example.com/radio.png" tvg-id="demo.radio" group-title="Radio",Radio Démo
https://example.com/audio/radio.mp3

#EXTINF:-1 tvg-name="VOD Démo" tvg-logo="https://example.com/vod.png" group-title="À la demande",VOD Démo
https://example.com/video/demo.mp4

Notes techniques

Limitations du navigateur.

XVB est soumis aux règles de sécurité web standard : CORS (les serveurs doivent autoriser les requêtes), HTTP/HTTPS (les flux non sécurisés sont bloqués sur un site HTTPS), et la compatibilité des codecs.

Utilisation responsable

Utilisez XVB avec des sources autorisées.

XVB est un outil neutre : il ne fournit aucun contenu. Vous êtes responsable des sources chargées. Évitez le piratage et utilisez uniquement des flux dont vous avez les droits d'accès.

Architecture.

Client-side intégral. Basé sur HLS.js, DASH.js et les API HTML5 native.
